# Adisapp Knowledge Base

## Proje Hakkında
Bu proje, Flutter kullanılarak geliştirilen **Adisapp** uygulamasıdır. Bu belge, proje yapısı, kurulum ve geliştirme süreçleri hakkında bilgi içerir.

## Başlangıç Rehberi (Getting Started)

### Gereksinimler
- Flutter SDK
- Dart SDK
- Android Studio veya VS Code (Flutter eklentileri ile)

### Uygulamayı Çalıştırma
Projeyi çalıştırmak için terminalde şu komutu kullanın:
```bash
flutter run
```

### Testleri Çalıştırma
Birim testlerini çalıştırmak için:
```bash
flutter test
```

## Proje Yapısı
- `lib/`: Uygulamanın kaynak kodları buradadır.
  - `main.dart`: Uygulamanın giriş noktası.
- `test/`: Test dosyaları.
- `android/`, `ios/`, `windows/`, `linux/`, `macos/`, `web/`: Platforma özel yapılandırma dosyaları.
- `pubspec.yaml`: Proje bağımlılıkları ve varlık tanımları.
